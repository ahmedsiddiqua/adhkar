# Adhkar
A webpage that you can add to your homescreen as a shortcut to remotely read your morning, evening and night (sleep) adhkars anywhere.

## 📋 Features
- Automatically alternates between morning, evening and night (sleep) adhkars depending on the time


## 🔗 Link
You can visit [adhkar.asterhyphen.xyz/](https://adhkar.asterhyphen.xyz/) to see how the shortcut looks like
